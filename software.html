Developing a Carbon Footprint Calculator Application

This application is designed to calculate the carbon footprint of vehicles and electrical appliances, providing a deeper understanding of greenhouse gas (GHG) emissions resulting from their usage.

It incorporates a comprehensive analysis of global warming potential (GWP), emission factors, and energy conversion as part of the carbon footprint assessment for both vehicles and household appliances. Therefore, this application aims to enhance awareness of the environmental impact of daily activities and encourage users to adopt more sustainable behaviors to reduce their carbon footprint.

Features

User Authentication

Provides secure login and registration for users.

Emission Calculation

Calculates CO₂ emissions based on usage.

Technologies Used

Java NetBeans IDE JDBC MySQL Java Swing
