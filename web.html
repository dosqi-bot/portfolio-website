Development of a multi-user e-commerce web application leveraging the CodeIgniter framework for efficient and scalable performance.

In the development of a web-based e-commerce application utilizing the CodeIgniter framework. The application is designed to facilitate online shopping and product management, incorporating two primary user roles: admin and customer. The admin role is responsible for managing the entire system, including adding, updating, deleting, and viewing products through a CRUD (Create, Read, Update, Delete) functionality. Meanwhile, the customer role enables users to browse available products, place orders, and manage their accounts efficiently.

Features

User Authentication

Secure login and registration for customers and admins.

Product Management

Admin can add, edit, delete, and manage product inventory.

Shopping Cart

Allows users to add and manage products in their cart.

Technologies Used

PHP CodeIgniter MySQL Bootstrap jQuery Ajax SweetAlert REST API
