Geospatial and Temporal Analysis of Disease Trends Across Indonesian Provinces

The development of a web-based system for monitoring disease trends enables authorities to track the spread of illnesses across various provinces in an interactive and visual manner. By utilizing data visualization tools such as Power BI and integrating geospatial analysis, this system provides real-time insights into disease distribution patterns. The collected data is processed and displayed through dynamic charts, graphs, and heatmaps, allowing health officials to quickly identify high-risk areas.

Furthermore, the system analyzes disease distribution across major islands such as Sumatra, Java, Kalimantan, Sulawesi, and Papua, offering comparative insights between regions. Historical data is used to detect patterns in disease progression, highlighting trends that may indicate emerging health risks. The platform also generates reports that visualize year-over-year changes in infection rates, assisting policymakers in formulating evidence-based health strategies. By enabling the comparison of disease prevalence between different areas, this system plays a crucial role in optimizing resource allocation and enhancing regional healthcare policies.

Features

Data Warehouse Integration

Stores and manages large-scale health data using SQL Server for efficient retrieval.

Trend Analysis & Forecasting

Analyzes historical disease data to predict future outbreaks.

Data Visualization

Generates interactive charts, heatmaps, and graphs for better insights.

Custom Data Filtering

Allows users to filter disease cases by location, disease type, and year.

Data Preprocessing

Checks for missing or inconsistent data and normalizes datasets for accurate analysis.

Predictive Analysis with Linear Regression

Uses linear regression models to forecast future disease trends and outbreak risks.

Technologies Used

Power BI SQL Server Python Jupyter Notebook Pandas HTML CSS Bootstrap
