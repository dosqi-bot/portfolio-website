Development of a mobile application that enables lecturers to efficiently input and manage student grades.

The development of a mobile application that enables lecturers to efficiently input and manage student grades plays a crucial role in enhancing academic administration. This application is specifically designed to support the digital transformation of academic activities, facilitating seamless interaction between lecturers and students. By providing a structured and user-friendly interface, the application streamlines the grading process, reducing manual workload and minimizing errors in grade recording. Furthermore, it ensures data security and accessibility, allowing lecturers to update and manage student performance records in real-time.

Features

User Authentication

Ensures secure login and role-based access for lecturers and students.

Student Grade Input

Allows lecturers to input and update student grades efficiently.

Grade Reports

Generates real-time student grade reports in an organized format.

Export & Backup

Allows lecturers to export student grades in various formats for record-keeping.

Role-Based Access Control

Restricts features based on user roles, ensuring data integrity.

Technologies Used

Kotlin Java Android Studio SQLite Jetpack Compose Material UI
